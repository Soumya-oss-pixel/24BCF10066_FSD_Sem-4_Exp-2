### 2.Overview

This project demonstrates the implementation of three interactive front-end components using HTML, CSS, and JavaScript. Each experiment focuses on a different aspect of user interaction and dynamic behavior in web applications.

The project is designed as a single navigable website with tab-based navigation for better usability and organization.


### 3.Learning Outcomes

After completing this experiment, the learner will be able to:

Understand the role of HTML, CSS, and JavaScript in building interactive web interfaces

Implement real-time DOM updates using JavaScript event handling

Design and manage single-page navigation without page reloads

Apply conditional rendering techniques for dynamic filtering

Work with SVG elements and handle mouse events for drawing operations

Improve user experience through responsive layouts and interactive feedback

Develop structured, modular, and readable front-end code
